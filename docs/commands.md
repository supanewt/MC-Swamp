# Rank Command Access

Generated from explicit LuckPerms grants in `plugins/LuckPerms/luckperms-h2-v2.mv.db` and matched against installed plugin commands where possible.

## Basic Users

These are the commands explicitly granted to the `default` group.

| Command | Description |
| --- | --- |
| `/balance` | Shows your current money balance. |
| `/betterrtp` | Randomly teleports you using BetterRTP. |
| `/rtp` | Short alias for random teleport. |
| `/rtp mining` | Randomly teleports you to the mining world. |
| `/dquests` | Opens or shows your daily quests. |
| `/delhome` | Deletes one of your saved homes. |
| `/home` | Teleports you to one of your homes. |
| `/ignore` | Blocks private messages from a player. |
| `/kit` | Opens or claims available kits. |
| `/kit claim` | Claims the `claim` kit. |
| `/kit food` | Claims the `food` kit. |
| `/kit pvp` | Claims the `pvp` kit. |
| `/kit tools` | Claims the `tools` kit. |
| `/list` | Shows online players. |
| `/msg` | Sends a private message. |
| `/msgtoggle` | Toggles whether you can receive private messages. |
| `/pay` | Sends money to another player. |
| `/quests` | Opens the quests menu or base quests command. |
| `/quests cancel` | Cancels a quest you have started. |
| `/quests start` | Starts a quest. |
| `/quests started` | Lists quests you already started. |
| `/quests track` | Tracks or untracks a quest objective. |
| `/realname` | Shows the real username behind a nickname. |
| `/sethome` | Creates a home location. |
| `/spawn` | Teleports you to server spawn. |
| `/suicide` | Instantly kills your player. |
| `/tags` | Opens or changes your chat tag. |
| `/tpaccept` | Accepts a teleport request. |
| `/tpa` | Requests teleport to another player. |
| `/tpahere` | Requests another player teleport to you. |
| `/tpdeny` | Denies a teleport request. |
| `/tptoggle` | Toggles whether players can send you teleport requests. |
| `/ultracosmetics toggle` | Toggles your cosmetics on or off. |
| `/warp` | Teleports to a public warp. |
| `/warp afk` | Teleports to the `afk` warp. |
| `/warp arena` | Teleports to the `arena` warp. |
| `/sell` | Sells items to the server shop. |
