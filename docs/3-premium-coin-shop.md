# What Players Can Buy With Coins

Last updated: 2026-04-01
Server: `MC Swamp`

This file documents the current configured ways players can spend `Coins` on the server.

Main access point:

- `/coinshop`

Main menu source:

- `plugins/DeluxeMenus/gui_menus/coinshop/coinshop_menu.yml`

## Main Coinshop Categories

Players can currently spend coins on:

- Crate keys
- Donator rank vouchers
- Command vouchers
- Furniture
- Toolskins
- Claimblocks
- Backpacks
- Pets
- Hats
- Particles
- Projectile effects
- Death effects
